# تمرین مشابه

## سوال

آرایه‌ای دایره‌ای `nums` داده شده.
برای هر عنصر، اولین عنصر بزرگ‌تر در سمت راست (با wrap-around) را پیدا کن.
اگر وجود ندارد `-1`.

## نمونه

- ورودی: `nums = [1,2,1]`
- خروجی: `[2,-1,2]`

## راهنمایی

1. با اندیس از `0` تا `2*n-1` برو (شبیه‌سازی دایره‌ای).
2. هنگام خواندن مقدار از `nums[i % n]` استفاده کن.
3. فقط اندیس‌های دور اول را push کن.
4. الگو همان monotonic decreasing stack است.

## هدف پیچیدگی

- زمان: `O(n)`
- حافظه: `O(n)`
