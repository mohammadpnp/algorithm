# تمرین مشابه

## سوال

سر لیست پیوندی و عدد `n` داده شده.
گره `n`-ام از انتهای لیست را حذف کن و head جدید را برگردان.

## نمونه

- ورودی: `1 -> 2 -> 3 -> 4 -> 5`, `n = 2`
- خروجی: `1 -> 2 -> 3 -> 5`

## راهنمایی

1. از dummy node قبل از head استفاده کن.
2. `fast` را `n` گام جلو ببر.
3. سپس `fast` و `slow` را همزمان حرکت بده تا `fast` به انتها برسد.
4. `slow.Next` همان گره هدف برای حذف است.

## هدف پیچیدگی

- زمان: `O(n)`
- حافظه: `O(1)`

## چالش اضافه

نسخه دو بار پیمایش (first count length) را هم پیاده‌سازی کن و مقایسه کن.
