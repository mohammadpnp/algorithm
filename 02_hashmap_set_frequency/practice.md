# تمرین مشابه

## سوال

آرایه `nums` و عدد `k` داده شده. بررسی کن آیا دو اندیس `i` و `j` وجود دارند که:

1. `nums[i] == nums[j]`
2. `|i - j| <= k`

اگر وجود داشت `true` وگرنه `false` برگردان.

## نمونه

- ورودی: `nums = [1,2,3,1], k = 3`
- خروجی: `true`

## راهنمایی

1. آخرین اندیس هر مقدار را در map نگه دار.
2. وقتی مقدار تکراری دیدی، فاصله اندیس را چک کن.
3. سپس اندیس فعلی را جایگزین کن.

## هدف پیچیدگی

- زمان: `O(n)`
- حافظه: `O(n)`

## تمرین تکمیلی

نسخه سخت‌تر: اگر به جای تساوی مقدار، اختلاف مقدارها هم شرط داشته باشند، چطور حل می‌کنی؟
